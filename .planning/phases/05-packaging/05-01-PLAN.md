## Goal
Create a Flatpak package that builds and runs Dupont with proper sandbox permissions.

Flatpak is the primary distribution format for GNOME and COSMIC. Users expect to find and install apps from Flathub or their software center. Without a Flatpak package, the app is only installable via `cargo install`.

       - `runtime`: `org.gnome.Platform`
       - `runtime-version`: `47`
       - `sdk`: `org.gnome.Sdk`
       - `command`: `dupont-app`
       - `finish-args`:
         - `--share=network` (fetch wallpapers from APIs)
         - `--filesystem=~/.config/dupont:create` (config persistence)
         - `--filesystem=~/.cache/dupont:create` (image cache)
         - `--socket=fallback-x11` (X11 compatibility)
         - `--device=dri` (GPU acceleration for GTK rendering)
       - `cleanup`: cleanup build artifacts after build
       - Build module: Rust with `rustc` and `cargo` from flathub shared-modules
       - Build steps: `cargo build --release -p dupont-app -p dupont-domain`
