# API Details from Research

## Bing Wallpaper of the Day API

**Endpoint:** `https://www.bing.com/HPImageArchive.aspx?format=js&idx=0&n=1&mkt=en-US`

**Parameters:**
- `format=js` - Return JSON format
- `idx=0` - Current day (0=today, 1=yesterday, etc.)
- `n=1` - Number of images to return
- `mkt=en-US` - Market region (e.g., en-US, zh-CN, ja-JP)

**Response Structure:**
```json
{
  "images": [
    {
      "startdate": "20240413",
      "fullstartdate": "202404130000",
      "enddate": "20240414",
      "url": "/th?id=OHR.SpringBlossom_EN-US123456&rs=1&c=4&h=300",
      "urlbase": "/th?id=OHR.SpringBlossom_EN-US123456",
      "copyright": "Cherry blossoms in Kyoto, Japan (© John Doe/Getty Images)",
      "copyrightlink": "https://www.bing.com/search?q=Kyoto+cherry+blossoms",
      "title": "Spring Blossoms",
      "quiz": "search?q=Bing+homepage+quiz",
      "wp": true,
      "hsh": "abc123",
      "drk": 1,
      "top": 1,
      "bot": 1,
      "hs": []
    }
  ]
}
```

**Image URL:** Full image URL is `https://www.bing.com{url}`

**Rate Limits:** ~1 request per second (implement backoff if needed)

## Microsoft Spotlight API

**Endpoint:** `https://arc.msn.com/v3/Delivery/Cache?pid=209567&fmt=json&rafb=0&ua=WindowsShellClient%2F0&disphorzres=1920&dispvertres=1080&lo=80217`

**Parameters:**
- `pid=209567` - Spotlight feature ID
- `fmt=json` - JSON format
- `rafb=0` - Random/featured balance
- `ua=WindowsShellClient/0` - User agent
- `disphorzres=1920` - Horizontal resolution
- `dispvertres=1080` - Vertical resolution
- `lo=80217` - Locale (80217 = en-US)

**Response Structure:**
```json
{
  "batchrsp": {
    "items": [
      {
        "item": [
          {
            "ad": {
              "imageFullscreen001": {
                "u": "https://www.bing.com/th?id=OHR.SpringBlossom_EN-US123456_1920x1080.jpg&rf=LaDigue_1920x1080.jpg&pid=hp",
                "w": 1920,
                "h": 1080,
                "sz": "524288",
                "o": "80217",
                "l": "https://www.bing.com/search?q=Spring+Blossoms&form=hpcapt",
                "t": "Spring Blossoms",
                "desc": "Cherry blossoms in Kyoto, Japan (© John Doe/Getty Images)"
              }
            }
          }
        ]
      }
    ]
  }
}
```

**Image URL:** Direct URL in `u` field

**Rate Limits:** Similar to Bing API (implement backoff if needed)

use async_trait::async_trait;
use reqwest::Client;
use serde::Deserialize;
use std::time::Duration;
use crate::source::Source;
use crate::wallpaper::Wallpaper;
use crate::error::SourceError;

/// Bing Wallpaper of the Day API response structure
#[derive(Debug, Deserialize)]
struct BingResponse {
    images: Vec<BingImage>,
}

#[derive(Debug, Deserialize)]
struct BingImage {
    #[serde(rename = "url")]
    url_path: String,
    #[serde(rename = "copyright")]
    description: String,
    #[serde(rename = "copyrightlink")]
    attribution_url: String,
    #[serde(rename = "title")]
    title: String,
    #[serde(rename = "startdate")]
    date: String,
}

/// Bing Wallpaper of the Day source
pub struct BingSource {
    client: Client,
    market: String,
    base_url: &'static str,
}

impl BingSource {
    /// Create a new BingSource with default market (en-US)
    pub fn new() -> Self {
        Self::with_market("en-US".to_string())
    }

    /// Create a new BingSource with specified market
    pub fn with_market(market: String) -> Self {
        Self {
            client: Client::builder()
                .timeout(Duration::from_secs(10))
                .build()
                .expect("Failed to create HTTP client"),
            market,
            base_url: "https://www.bing.com/HPImageArchive.aspx",
        }
    }
}

impl Default for BingSource {
    fn default() -> Self {
        Self::new()
    }
}

#[async_trait]
impl Source for BingSource {
    async fn fetch(&self) -> Result<Wallpaper, SourceError> {
        // Build API URL with parameters
        let url = format!(
            "{}?format=js&idx=0&n=1&mkt={}",
            self.base_url, self.market
        );

        // Fetch from Bing API with retry for rate limiting
        let response = self.fetch_with_retry(&url, 3).await?;

        // Parse JSON response
        let bing_response: BingResponse = response.json().await
            .map_err(|e| SourceError::ParseError(format!("Failed to parse Bing API response: {}", e)))?;

        // Extract first image
        let image = bing_response.images.into_iter()
            .next()
            .ok_or(SourceError::NoWallpaperFound)?;

        // Construct full image URL
        let full_url = format!("https://www.bing.com{}", image.url_path);

        // Extract attribution (copyright holder from copyright text)
        let attribution = extract_attribution(&image.description);

        // Create ID from date
        let id = format!("bing-{}", image.date);

        Ok(Wallpaper {
            id,
            url: full_url,
            title: image.title,
            description: image.description,
            attribution,
            source: "bing".to_string(),
            thumbnail_url: None, // Can add thumbnail URL if Bing provides it
        })
    }

    fn id(&self) -> &'static str {
        "bing"
    }

    fn name(&self) -> &'static str {
        "Bing Wallpaper of the Day"
    }
}

impl BingSource {
    async fn fetch_with_retry(&self, url: &str, max_retries: u32) -> Result<reqwest::Response, SourceError> {
        let mut retries = 0;

        loop {
            let response = self.client.get(url).send().await
                .map_err(|e| SourceError::HttpError(e))?;

            let status = response.status();

            // Check for rate limiting
            if status.as_u16() == 429 {
                retries += 1;
                if retries >= max_retries {
                    return Err(SourceError::RateLimited {
                        source: "Bing".to_string(),
                    });
                }

                // Exponential backoff: 1s, 2s, 4s
                let backoff = Duration::from_secs(2u64.pow(retries));
                tokio::time::sleep(backoff).await;
                continue;
            }

            // Return response for non-rate-limit status codes
            if !status.is_success() {
                return Err(SourceError::Unavailable {
                    source: format!("Bing API returned status {}", status),
                });
            }

            return Ok(response);
        }
    }
}

/// Extract attribution (copyright holder) from copyright string
fn extract_attribution(copyright: &str) -> String {
    // Format: "Description text (© Copyright Holder)"
    copyright
        .split("©")
        .nth(1)
        .and_then(|s| s.split(')').next())
        .map(|s| s.trim().to_string())
        .unwrap_or_else(|| "Microsoft Bing".to_string())
}

use reqwest::Client;
use serde::Deserialize;
use std::time::Duration;
use crate::source::Source;
use crate::wallpaper::Wallpaper;
use crate::error::SourceError;

/// Microsoft Spotlight API response structure (deeply nested)
#[derive(Debug, Deserialize)]
struct SpotlightResponse {
    batchrsp: SpotlightBatch,
}

#[derive(Debug, Deserialize)]
struct SpotlightBatch {
    items: Vec<SpotlightItem>,
}

#[derive(Debug, Deserialize)]
struct SpotlightItem {
    item: Vec<SpotlightItemData>,
}

#[derive(Debug, Deserialize)]
struct SpotlightItemData {
    #[serde(rename = "ad")]
    ad: SpotlightAd,
}

#[derive(Debug, Deserialize)]
struct SpotlightAd {
    #[serde(rename = "imageFullscreen001")]
    image: SpotlightImage,
}

#[derive(Debug, Deserialize)]
struct SpotlightImage {
    #[serde(rename = "u")]
    url: String,
    #[serde(rename = "t")]
    title: String,
    #[serde(rename = "desc")]
    description: String,
    #[serde(rename = "l")]
    attribution_url: String,
}

/// Microsoft Spotlight source
pub struct SpotlightSource {
    client: Client,
    locale: String,
    resolution: (u32, u32),
    base_url: &'static str,
}

impl SpotlightSource {
    /// Create a new SpotlightSource with default settings
    pub fn new() -> Self {
        Self::with_locale("80217".to_string()) // 80217 = en-US
    }

    /// Create a new SpotlightSource with specified locale code
    pub fn with_locale(locale: String) -> Self {
        Self {
            client: Client::builder()
                .timeout(Duration::from_secs(10))
                .build()
                .expect("Failed to create HTTP client"),
            locale,
            resolution: (1920, 1080), // Default Full HD resolution
            base_url: "https://arc.msn.com/v3/Delivery/Cache",
        }
    }

    /// Set resolution for fetched wallpapers
    pub fn with_resolution(mut self, width: u32, height: u32) -> Self {
        self.resolution = (width, height);
        self
    }
}

impl Default for SpotlightSource {
    fn default() -> Self {
        Self::new()
    }
}

#[async_trait]
impl Source for SpotlightSource {
    async fn fetch(&self) -> Result<Wallpaper, SourceError> {
        // Build API URL with parameters
        let url = format!(
            "{}?pid=209567&fmt=json&rafb=0&ua=WindowsShellClient%2F0&disphorzres={}&dispvertres={}&lo={}",
            self.base_url, self.resolution.0, self.resolution.1, self.locale
        );

        // Fetch from Spotlight API with retry for rate limiting
        let response = self.fetch_with_retry(&url, 3).await?;

        // Parse JSON response
        let spotlight_response: SpotlightResponse = response.json().await
            .map_err(|e| SourceError::ParseError(format!("Failed to parse Spotlight API response: {}", e)))?;

        // Extract image from deeply nested structure
        let image = spotlight_response
            .batchrsp
            .items
            .into_iter()
            .filter_map(|item| item.item.into_iter().next())
            .map(|data| data.ad.image)
            .next()
            .ok_or(SourceError::NoWallpaperFound)?;

        // Create ID from URL hash (simple approach)
        let id = format!("spotlight-{}", hash_url(&image.url));

        // Extract attribution from description (format: "Description (© Copyright)")
        let attribution = extract_attribution(&image.description);

        Ok(Wallpaper {
            id,
            url: image.url,
            title: image.title,
            description: image.description,
            attribution,
            source: "spotlight".to_string(),
            thumbnail_url: None,
        })
    }

    fn id(&self) -> &'static str {
        "spotlight"
    }

    fn name(&self) -> &'static str {
        "Microsoft Spotlight"
    }
}

/// Simple hash function to generate ID from URL
fn hash_url(url: &str) -> String {
    use std::collections::hash_map::DefaultHasher;
    use std::hash::{Hash, Hasher};

    let mut hasher = DefaultHasher::new();
    url.hash(&mut hasher);
    format!("{:x}", hasher.finish())
}

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Spoofing | HTTP requests to Bing/Spotlight APIs | mitigate | Validate URLs before requests, enforce HTTPS, use reqwest's default TLS verification |
| T-01-02 | Tampering | Bing API response parsing | mitigate | Use strict serde schema validation with `#[serde(rename)]` for all fields, deny unknown fields if possible |
| T-01-03 | Tampering | Spotlight API response parsing | mitigate | Use strict serde schema validation for deeply nested structure, handle missing fields gracefully with `?` operator |
| T-01-04 | Information disclosure | Error messages | mitigate | Avoid exposing API keys or internal paths in error messages, return generic error text for public-facing errors |
| T-01-05 | Denial of Service | Rate limiting | mitigate | Implement exponential backoff (1s, 2s, 4s) on HTTP 429 responses, max 3 retries before returning RateLimited error |
| T-01-06 | Denial of Service | Slow network responses | mitigate | Set 10-second timeout on HTTP client via reqwest::Client::builder().timeout() |
